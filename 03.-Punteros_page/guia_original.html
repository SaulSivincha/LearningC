Estructura base de un programa C

Todo programa C ejecutable necesita una funcion de entrada, normalmente main.

Tu archivo 1.HolaMundo.c contiene la forma minima para imprimir en pantalla.

1.HolaMundo.c
#include <stdio.h>

int main() {
    printf("Hola mundo");
    return 0;
}

Lectura linea por linea

  • #include <stdio.h>: pide declaraciones de entrada/salida estandar, como printf y scanf.
  • int main(): funcion donde empieza el programa. Retorna un entero al sistema operativo.
  • { ... }: bloque de instrucciones.
  • printf(...);: llamada a funcion. La instruccion termina con ;.
  • return 0;: final correcto del programa.
Sintaxis Significado Ejemplo
#include <archivo.h> Incluye declaraciones de una biblioteca. #include <stdio.h>
tipo nombre(...) Declara una funcion. int main()
{ } Agrupa instrucciones en un bloque. if (edad >= 18) { ... }
; Cierra una sentencia simple. int edad = 20;
// y /* */ Comentarios ignorados por el compilador. // una linea

Variables y tipos

Una variable es una zona de memoria con nombre, tipo y valor.

2.Variables.c
#include <stdio.h>

int main() {
    int edad = 20;
    char nombre[] = "Saul";
    double estatura = 1.71;

    printf("%d", edad);
    printf("%.2f", estatura);
    printf("%s", nombre);
    return 0;
}

Partes de una variable

  • int edad = 20;: entero.
  • double estatura = 1.71;: numero real de doble precision.
  • char nombre[] = "Saul";: arreglo de caracteres terminado en '\0'.
  • =: asignacion, no comparacion.
Tipo Uso comun Formato con printf Formato con scanf
int Enteros: edad, cantidad, indice. %d %d con &variable
double Decimales: estatura, promedio. %f, %.2f %lf con &variable
char Un caracter. %c %c con &variable
char[] String guardado en arreglo. %s %29s sin & si el buffer es char nombre[30]

sizeof(variable) devuelve cuantos bytes ocupa ese objeto en memoria. En arreglos sirve para calcular cantidad de elementos, pero en punteros devuelve el tamano del puntero, no del arreglo original.

printf y cadenas de formato

printf imprime texto normal y valores formateados.

El primer argumento de printf es una cadena de formato. Los caracteres normales salen tal cual. Las conversiones empiezan con %.

Salida formateada
int edad = 20;
double estatura = 1.71;
char nombre[] = "Saul";

printf("Nombre: %s\n", nombre);
printf("Edad: %d\n", edad);
printf("Estatura: %.2f\n", estatura);
Formato Tipo esperado Ejemplo
%d int printf("%d", edad);
%f double en printf printf("%f", estatura);
%.2f double, con 2 decimales visibles printf("%.2f", 1.714);
%s char * apuntando a string terminado en '\0' printf("%s", nombre);
%c int que representa un caracter printf("%c", inicial);
%p void *, una direccion printf("%p", (void *)&edad);
%% Ningun argumento printf("100%%");

En C, el formato y el tipo deben coincidir. Si escribes printf("%d", estatura), el compilador puede advertirlo y el programa puede imprimir basura.

Mal vs bien
double estatura = 1.71;

printf("%d\n", estatura);     // mal: %d espera int
printf("%.2f\n", estatura);   // bien: %.2f imprime double con 2 decimales

scanf y entradas

scanf lee datos desde la entrada estandar y necesita direcciones donde guardar.

3.Leer_Entradas.c mejorado
#include <stdio.h>

int main() {
    int edad;

    printf("Escribe tu edad: ");

    if (scanf("%d", &edad) != 1) {
        printf("Entrada invalida\n");
        return 1;
    }

    printf("Tu edad es %d\n", edad);
    return 0;
}

Por que va &edad

edad es el valor. &edad es la direccion donde vive ese valor. scanf no puede modificar tu variable si solo le das una copia del valor; necesita saber donde escribir.

Esta es la puerta de entrada a punteros.

Reglas practicas de scanf
  • Para int: scanf("%d", &edad).
  • Para double: scanf("%lf", &estatura).
  • Para char: scanf(" %c", &letra). El espacio antes de %c consume saltos de linea pendientes.
  • Para char nombre[30]: scanf("%29s", nombre). No uses &nombre con %s.
  • Verifica el retorno: si querias leer 1 dato, espera que retorne 1.

scanf("%s", nombre) lee hasta el primer espacio. Para nombres con espacios, conviene fgets. Para aprender punteros esta bien usar scanf, pero con ancho maximo: %29s si el buffer mide 30.

Operadores

Los operadores combinan valores. Algunos producen numeros, otros producen verdadero/falso.

4.Operaciones_Matematicas.c mejorado
#include <stdio.h>

int main() {
    int a;
    int b;

    printf("Ingresa el primer numero: ");
    scanf("%d", &a);

    printf("Ingresa el segundo numero: ");
    scanf("%d", &b);

    printf("Suma: %d\n", a + b);
    printf("Resta: %d\n", a - b);
    printf("Multiplicacion: %d\n", a * b);

    if (b != 0) {
        printf("Division entera: %d\n", a / b);
        printf("Residuo: %d\n", a % b);
    } else {
        printf("No se puede dividir entre cero\n");
    }

    return 0;
}

Tabla rapida

OperadorUso
+Suma.
-Resta o signo negativo.
*Multiplicacion. Tambien dereferencia en punteros.
/Division. Si ambos son int, el resultado es entero.
%Residuo de division entera.
++Incremento: i++.
--Decremento: i--.

En int a = 7; int b = 2;, a / b da 3, no 3.5. Para decimales, al menos uno debe ser double: (double)a / b.

Strings

En C, un string no es un tipo especial: normalmente es un arreglo de char terminado en '\0'.

6.Strings.c corregido
#include <stdio.h>
#include <string.h>

int main() {
    char nombre[30];

    printf("Ingresa tu nombre: ");

    if (scanf("%29s", nombre) != 1) {
        printf("No se pudo leer el nombre\n");
        return 1;
    }

    if (strcmp(nombre, "Daniel") == 0) {
        printf("Hola, %s\n", nombre);
        printf("Coincidencia exacta\n");
    } else {
        printf("Hola, %s\n", nombre);
    }

    printf("Tu nombre tiene %zu caracteres visibles\n", strlen(nombre));

    return 0;
}

Conceptos clave

  • char nombre[30]: reserva espacio para hasta 29 caracteres mas '\0'.
  • %29s: limita la lectura para no desbordar el arreglo.
  • strlen(nombre): cuenta bytes antes del '\0'.
  • strcmp(a, b) == 0: las cadenas son iguales.
  • %zu: formato correcto para imprimir size_t, el tipo que retorna strlen.
Representacion de "Saul" en memoria
Indice 0 1 2 3 4
Contenido 'S' 'a' 'u' 'l' '\0'

El caracter nulo no se imprime, pero marca donde termina el string. Si falta, funciones como strlen o printf("%s") seguirian leyendo memoria hasta encontrar un cero por casualidad.

Tu condicion original strlen(nombre) <= 0 && strcmp(nombre, "Daniel") == 0 no puede cumplirse para "Daniel": si el nombre es Daniel, su longitud no es 0. Para igualdad usa directamente strcmp(nombre, "Daniel") == 0.

Arreglos

Un arreglo guarda varios elementos del mismo tipo en memoria consecutiva.

Arreglo de enteros
#include <stdio.h>

int main() {
    int numeros[] = {10, 20, 30, 40, 50};
    int cantidad = sizeof(numeros) / sizeof(numeros[0]);

    for (int i = 0; i < cantidad; i++) {
        printf("numeros[%d] = %d\n", i, numeros[i]);
    }

    return 0;
}

Reglas

  • Los indices empiezan en 0.
  • Si hay 5 elementos, el ultimo indice es 4.
  • numeros[i] accede al elemento i.
  • sizeof(numeros) / sizeof(numeros[0]) calcula elementos solo mientras numeros siga siendo arreglo, no puntero.

Los arreglos conectan directamente con punteros: en muchas expresiones, el nombre del arreglo se convierte en direccion del primer elemento. Por eso int *p = numeros; apunta a numeros[0].

Bucles

Los bucles repiten instrucciones. Ya aparece un for en tu archivo de punteros.

for

Ideal cuando sabes cuantas veces repetir.

for
for (int i = 0; i < 5; i++) {
    printf("%d\n", i);
}

while

Repite mientras la condicion siga siendo verdadera.

while
int i = 0;
while (i < 5) {
    printf("%d\n", i);
    i++;
}

do while

Ejecuta al menos una vez.

do while
int opcion;
do {
    scanf("%d", &opcion);
} while (opcion != 0);

Punteros

Un puntero es una variable que guarda una direccion de memoria.

Resumen mental: &x pregunta "donde esta x?". *p pregunta "que valor hay donde apunta p?".

Direccion y dereferencia
#include <stdio.h>

int main() {
    int edad = 20;
    int *p = &edad;

    printf("edad = %d\n", edad);
    printf("&edad = %p\n", (void *)&edad);
    printf("p = %p\n", (void *)p);
    printf("*p = %d\n", *p);

    *p = 21;
    printf("edad ahora = %d\n", edad);

    return 0;
}

Que significa cada parte

  • int *p: p guarda direcciones de variables int.
  • &edad: direccion de edad.
  • p = &edad: el puntero apunta a edad.
  • *p: valor que esta en la direccion guardada en p.
  • *p = 21: modifica edad a traves de su direccion.
Codigo Nombre tecnico Idea humana
&variable Operador direccion Dame donde vive esta variable.
tipo *p Declaracion de puntero Esta variable guarda una direccion hacia un tipo.
*p Dereferencia Entra a la direccion y usa el valor que hay ahi.
NULL Puntero nulo No apunta a ningun objeto valido.

Tu idea de 7.Punteros.c apunta a esto: recorrer un arreglo usando un puntero.

7.Punteros.c corregido
#include <stdio.h>

int main() {
    int numeros[] = {10, 20, 30, 40, 50};
    int cantidad = sizeof(numeros) / sizeof(numeros[0]);

    int *puntero = numeros;

    for (int i = 0; i < cantidad; i++) {
        printf("Valor: %d | Direccion: %p\n",
               *(puntero + i),
               (void *)(puntero + i));
    }

    return 0;
}

Equivalencias

Con arregloCon puntero
numeros[0]*puntero
numeros[i]*(puntero + i)
&numeros[i]puntero + i

Aritmetica de punteros

Si puntero es int *, entonces puntero + 1 no avanza 1 byte. Avanza al siguiente int. Si un int ocupa 4 bytes en tu maquina, la direccion sube 4 bytes.

Los punteros permiten que una funcion modifique variables externas. Eso ya lo hace scanf.

Modificar por direccion
#include <stdio.h>

void cumplir_anios(int *edad) {
    if (edad != NULL) {
        *edad = *edad + 1;
    }
}

int main() {
    int edad = 20;

    cumplir_anios(&edad);

    printf("Edad: %d\n", edad);
    return 0;
}

Sin puntero, la funcion recibe una copia. Con puntero, recibe la direccion de la variable real.

char nombre[] = "Saul";

Crea un arreglo modificable con copia de los caracteres.

Arreglo modificable
char nombre[] = "Saul";
nombre[0] = 'P';
printf("%s\n", nombre); // Paul

char *nombre = "Saul";

El puntero apunta a un literal de string. No debes modificar sus caracteres.

Literal no modificable
char *nombre = "Saul";
printf("%s\n", nombre);

// nombre[0] = 'P'; // no lo hagas

Esta diferencia importa: arreglo y puntero se parecen al imprimir con %s, pero no tienen las mismas reglas de almacenamiento ni modificacion.

Errores peligrosos

  • Usar un puntero sin inicializar.
  • Dereferenciar NULL.
  • Salir del limite de un arreglo: numeros[5] si solo hay 5 elementos.
  • Guardar la direccion de una variable local y usarla fuera de su vida.
  • Confundir p con *p.

Habitos defensivos

  • Inicializa punteros: int *p = NULL;.
  • Antes de *p, asegurate de que p != NULL.
  • Para imprimir direcciones, usa %p con cast a void *.
  • Compila con -Wall -Wextra.
  • Usa nombres claros: puntero, ptr, actual, inicio.

Laboratorio de memoria

Mueve el indice para ver como cambia puntero + i.

Expresion actual

*(puntero + 0) lee 10.

Codigo equivalente

Si la direccion base fuera 0x1000 y cada int ocupa 4 bytes, puntero + 0 seria 0x1000.
Modelo usado
int numeros[] = {10, 20, 30, 40, 50};

Errores detectados en tus archivos

No son fracasos: son exactamente los errores que conviene entender antes de avanzar.

6.Strings.c: scanf("%29s", &nombre)

Para char nombre[30], el nombre del arreglo ya se convierte en direccion del primer caracter cuando se pasa a scanf. Usa:

Correccion
scanf("%29s", nombre);

&nombre tiene otro tipo: direccion del arreglo completo, no direccion de char. Puede parecer que funciona, pero no es la forma correcta para %s.

6.Strings.c: condicion imposible con strlen y strcmp
Original problematico
if (strlen(nombre) <= 0 && strcmp(nombre, "Daniel") == 0) {
    ...
}

Si nombre es "Daniel", su longitud es 6, no 0. Para comparar con Daniel:

Correccion
if (strcmp(nombre, "Daniel") == 0) {
    printf("Hola, %s\n", nombre);
}
